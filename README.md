Welcome to the official translation repository for HTTP Tools.

This repository contains all language files used by the HTTP Tools mobile application. We welcome contributions from volunteers around the world who would like to help translate the application into their native language.

How to Contribute

1. Fork this repository.
2. Create a new branch for your changes.
3. Translate or improve the language file.
4. Submit a Pull Request.

Please only modify translation files unless requested otherwise.

Translation Guidelines

* Keep the original meaning of each text.
* Use natural and common expressions in your language.
* Do not translate placeholders such as %s, %d, {name}, {count}, or HTML tags.
* Keep URLs, file names, and code snippets unchanged.
* If you are unsure about a translation, leave a comment in your Pull Request.

Language Files

Each language is stored as a separate JSON file.

Example:
lang/
├── en.json      # English (source language)
├── id.json      # Indonesian
├── fr.json      # French
├── es.json      # Spanish
├── ja.json      # Japanese
└── ...

English (en.json) is the source language. All other translations should follow the same keys.

Reporting Issues

If you find incorrect translations, missing text, or formatting problems, please open an Issue or submit a Pull Request.
